# 持久层 - 状态管理

## 核心技术

### Cookie 池管理
- **Cookie 类型**：
  - 指纹 Cookie：长期标识（如 _ga、_gid）
  - 会话 Cookie：短期登录状态
  - 功能 Cookie：偏好设置

- **Cookie 属性**：
  - Secure：仅 HTTPS 传输
  - HttpOnly：禁止 JS 访问
  - SameSite：跨域控制
  - Domain/Path：作用域限制
  - Max-Age/Expires：过期时间

### 浏览器 Profile 管理
- **User Data 目录**：
  - 存储位置：自定义路径
  - 包含内容：Cookie、LocalStorage、缓存、历史
  - 隔离策略：每个账号独立 Profile

- **生命周期**：
  1. 创建 Profile
  2. 首次访问建立指纹
  3. 保存状态
  4. 后续复用恢复

### 状态同步与回滚
- **状态快照**：
  - Cookie 序列化
  - LocalStorage/SessionStorage 备份
  - IndexedDB 导出

- **恢复策略**：
  - Session 过期自动恢复
  - 失败状态回滚
  - 多状态版本管理

### IndexedDB 管理
- **存储操作**：
  - 读取：from()、getAll()
  - 写入：add()、put()
  - 删除：delete()、clear()

- **应用场景**：
  - 大量结构化数据存储
  - 离线应用数据
  - Web 应用状态
