# 反反爬虫技能树 - 总览

> 网络爬虫与反爬虫是一场持续的军备竞赛。本文系统整理反反爬虫技术体系，构建完整的技能树。

## 技能树全景

```
反反爬虫技能树
│
├── [[01-基础层]]（必学）
│   ├── HTTP 请求基础
│   ├── 浏览器工作原理
│   ├── 指纹技术原理
│   └── 请求管理
│
├── [[02-环境层]]（核心）
│   ├── 浏览器指纹伪装
│   ├── 网络环境隔离
│   ├── 设备信息伪造
│   └── 反调试技术
│
├── [[03-行为层]]（进阶）
│   ├── 鼠标/键盘行为模拟
│   ├── 浏览器操作轨迹
│   └── 人类行为模式
│
├── [[04-持久层]]（高级）
│   ├── Cookies/Session 管理
│   ├── 浏览器指纹持久化
│   └── 状态同步
│
├── [[05-验证层]]（专攻）
│   ├── 验证码识别（OCR/ML）
│   ├── 人机验证绕过
│   └── 行为验证对抗
│
└── [[06-架构层]]（精通）
    ├── 分布式爬虫架构
    ├── 反检测策略库
    ├── AI 智能化
    └── 机器学习（ML）
```

## 学习路径

### 入门阶段（1-2 周）
**目标**：掌握基础层技术
- [[01-基础层]]
- 学习 HTTP 协议
- 理解浏览器工作原理
- 掌握请求头构造

### 进阶阶段（1-2 个月）
**目标**：掌握环境层和行为层
- [[02-环境层]]
- [[03-行为层]]
- Navigator 对象伪造
- Canvas/WebGL 指纹处理
- 鼠标键盘行为模拟

### 高级阶段（3-6 个月）
**目标**：掌握持久层和验证层
- [[04-持久层]]
- [[05-验证层]]
- Cookie 池管理
- 验证码识别
- Session 持久化

### 精通阶段（持续）
**目标**：掌握架构层
- [[06-架构层]]
- 分布式架构
- AI 智能体
- 机器学习应用

## 检测与对抗的本质

| 维度 | 网站侧（检测） | 爬虫侧（反检测） |
|------|--------------|-----------------|
| **目标** | 识别自动化访问，保护数据/资源 | 模拟真实用户行为，获取目标数据 |
| **手段** | 指纹识别、行为分析、环境检测 | 指纹伪造、行为模拟、环境隔离 |
| **技术栈** | JS 检测、服务器检测、设备指纹 | 浏览器自动化、代理池、验证码识别 |
