## 核心技术

### 分布式架构设计

```
分布式爬虫架构
│
├── Master 节点（任务调度）
│   ├── 任务队列（Redis）
│   ├── 策略管理
│   ├── 代理池管理
│   └── 监控面板
│
├── Worker 节点（执行任务）
│   ├── 任务拉取
│   ├── 反检测策略应用
│   ├── 数据采集
│   └── 结果上传
│
└── 数据存储
    ├── Redis（状态/缓存）
    ├── MongoDB（原始数据）
    └── Elasticsearch（搜索）
```

### 反检测策略库

- **策略分类**：
  - 基础策略：简单请求
  - 标准策略：中等伪装
  - 高级策略：完整指纹
  - 极限策略：AI 行为模拟

- **策略路由**：
  ```python
  def select_strategy(url, risk_level):
      if risk_level == "low":
          return "basic"
      elif risk_level == "medium":
          return "standard"
      elif risk_level == "high":
          return "advanced"
      else:
          return "ultimate"
  ```

### 速率限制算法

**令牌桶算法**
- 固定容量桶
- 恒定速率补充令牌
- 请求消耗令牌

**滑动窗口算法**
- 固定时间窗口
- 窗口内请求计数
- 超限则等待

**自适应限流**
- 根据响应动态调整
- 检测时增加延迟
- 成功时适当加速

### AI 智能体

**强化学习决策**
- 状态：成功率、被检测次数、请求间隔
- 动作：：增加延迟、减少延迟、切换代理、重试、放弃
- 奖励：：成功 +1，失败 -1

**自适应导航**
- 根据页面动态调整策略
- 检测到验证码时切换策略
- 失败时尝试不同的检测绕过

**动态选择器**
- 智能生成选择器
- 自动验证选择器有效性
- 失败时自动回退

### 机器学习应用

**特征工程**
- 页面特征：HTML 结构、元素数量
- 请求特征：头部、频率、时序
- 行为特征：鼠标轨迹、打字节奏

**模式识别**
- 反爬虫机制检测
- 异常访问模式识别
- 验证码类型分类

**预测模型**
- 风险预测：预测访问风险
- 趋势分析：成功率趋势
- 资源消耗预测

**智能分类**
- 网站类型自动分类
- 反检测策略分类
- 代理质量分类
