## 核心技术

### 验证码类型与应对

| 验证码类型 | 检测原理 | 应对策略 | 难度 |
|-----------|---------|----------|------|
| 文本验证码 | 图像文字识别 | OCR/CNN | ⭐⭐ |
| 滑动验证码 | 轨迹行为分析 | 轨迹生成 | ⭐⭐⭐ |
| 点选验证码 | 目标检测 | YOLO/模板匹配 | ⭐⭐⭐⭐ |

| reCAPTCHA v3 | 风险评分 | 降低风险特征 | ⭐⭐⭐⭐⭐ |
| 行为验证码 | 综合行为 | 完整模拟 | ⭐⭐⭐⭐⭐ |

### OCR 识别方案

**Tesseract OCR**
- 优势：开源免费
- 劣势：准确率较低，需要预处理
- 适用：简单文本验证码

```python
import pytesseract
from PIL import Image
import cv2

# 图像预处理
img = cv2.imread("captcha.png")
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
_, binary = cv2.threshold(gray, 127, 255, cv2.THRESH_BINARY)

# OCR 识别
text = pytesseract.image_to_string(binary)
```

**CNN 模型**
- 优势：准确率高
- 劣势：需要训练数据
- 适用：复杂验证码

### 滑动验证码破解

**模板匹配**
- 找出缺口位置
- 计算滑动距离
- 生成滑动轨迹

```python
# 缺口检测
def find_gap(bg, slice):
    result = cv2.matchTemplate(bg, slice, cv2.TM_CCOEFF_NORMED)
    min_val, max_val, min_loc, max_loc = cv2.minMaxLoc(result)
    return max_loc
```

**轨迹生成**
- 贝塞尔曲线模拟
- 速度变化（加速-减速）
- 回退抖动

### reCAPTCHA 对抗

**reCAPTCHA v2**
- 点击复选框
- 图像选择
- 使用 2captcha 等打码平台

**reCAPTCHA v3**
- 降低风险评分
- 模拟正常用户行为
- 高质量代理
- 长时间 Session

### 行为验证码完整模拟

- 鼠标移动轨迹
- 点击行为
- 停留时间
- 滚动行为
- 表单填写节奏
