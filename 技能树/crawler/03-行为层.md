## 核心技术

### 鼠标行为模拟
- **贝塞尔曲线轨迹**：
  - 三阶贝塞尔曲线生成平滑路径
  - 随机控制点增加自然感
  - 速度变化模拟加速/减速

- **点击位置偏移**：
  - 非中心点击：元素中心 ± 随机偏移
  - 模拟不精确的点击行为

- **鼠标抖动**：
  - 点击前微小随机移动
  - 模拟手部轻微抖动

### 键盘行为模拟
- **打字速度与节奏**：
  - 平均 200-300ms/字符
  - 随机延迟模拟真实打字
  - 功能键延迟更长

- **事件完整性**：
  - keydown → keypress → keyup
  - 正确的 keyCode 和 which

- **键盘修正行为**：
  - 随机打错字符
  - 删除重打模拟

### 滚动行为模拟
- **分段滚动**：
  - 分多次滚动而非一次性到底
  - 每次滚动 2-5 屏高度
  - 随机停留 1-3 秒

- **惯性模拟**：
  - 逐渐减速的滚动
  - 模拟真实滚动物理效果

### 视线停留时间
- **不同元素类型停留时间**：
  - 标题：2-4 秒
  - 正文段落：5-15 秒
  - 图片：3-8 秒
  - 链接：1-3 秒

## 行为模式示例

### 完整交互流程
1. 页面加载 → 等待 2-5 秒
2. 随机滚动浏览 → 分 3-5 次滚动
3. 停留在兴趣点 → 5-15 秒
4. 移动鼠标到目标 → 贝塞尔曲线
5. Hover 停留 → 0.5-2 秒
6. 点击 → 随机偏移
7. 表单填写 → 随机打字速度
8. 提交前检查 → 停顿 1-3 秒
9. 提交 → 自然延迟
